Xiaomi Bootloader Unlock Tool

Step-by-Step Installation Guide

Step 1 — Install Required App

Download and install Termux from F-Droid:

https://f-droid.org/packages/com.termux/

Open the Termux app after installation.

Step 2 — Update Termux

Run the following command:

pkg update && pkg upgrade -y

Step 3 — Install Unlock Tool

Paste this command inside Termux:

curl -fsSL https://aghavlabs.github.io/BootloaderUnlockXiaomi/install.sh | bash

Step 4 — Start the Tool

After installation run:

unlock

Step 5 — Login to Mi Account

  • Enter your Mi Account ID
  • Enter your Mi Account Password
  • Use the same account logged into your Xiaomi phone

Important Notes

  • Mi account must be older than 30 days
  • Your device must be linked to the Mi account
  • Leave Termux open if the tool is waiting for quota reset
  • Unlock permission depends on Xiaomi server approval
