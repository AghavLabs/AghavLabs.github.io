Xiaomi Bootloader Unlock Tool Guide

This guide explains how to install and run the Xiaomi Bootloader Unlock Tool using Termux.

Step 1 — Install Termux

Download and install Termux from F-Droid:

https://f-droid.org/packages/com.termux/

Install the app and open it.

Step 2 — Update Termux

Run the following command:

pkg update && pkg upgrade -y

Step 3 — Install Unlock Tool

Paste this command in Termux:

curl -fsSL https://aghavlabs.github.io/BootloaderUnlockXiaomi/install.sh | bash

Step 4 — Run the Tool

After installation run:

unlock

Step 5 — Login to Mi Account

The tool will ask for:

  • Mi Account ID
  • Mi Account Password

Enter the Mi account that is linked to your Xiaomi device.

Important Notes

  • Your Mi account must be older than 30 days.
  • Use the same account that is logged into your Xiaomi phone.
  • If unlock quota is reached, try again later.
